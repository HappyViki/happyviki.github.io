About Lessons

I'm using the project-based approach to help kids see their programming results right away and stay interested. My theory is that if projects contain a subject that the kid enjoys, then it'll be easier for them to continue being interested in learning more.

  • Scratch is drag-n-drop, so it's easy to see results right away.
  • Python is easy to read, so I use it as an introductory typing language.
  • JavaScript can be programmed right in the browser, so I use it to show how to interact with the web.
  • mBot2 is a robot that I bought with lots of fun sensors that can be programmed with drag-n-drop or Python. I'm excited to teach robotics with it :)
