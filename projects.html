ESP8266 LED Pet

Once you connect it to a power source and wifi, you can feed it carrots through the browser on your computer.

I used this project to learn C++ and ESP8266 programming. I learned about LED light manipulation and memory management through this project.

I made this with the help of my friends at 801 Labs Hackerspace, a LinkedIn Learning course, and googling keywords. I haven't done a C++ project like this before, it was fun learning a new language to build something cool.
